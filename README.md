# dedup
Dedup Challenge

Ruby Version: Ruby 2.5.1

## Install Dependencies
```
#text 1.3.1
gem install text

#sinatra 2.0.5
gem install sinatra

#sinatra-contrib 2.0.5
gem install sinatra-contrib
```


## Start up

```
ruby dedup.rb
```
Open up browser to : http://localhost:4567/
Result should follow
